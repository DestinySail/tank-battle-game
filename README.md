# 坦克大战 - Tank Battle

一个使用 Vue3 + TypeScript + Canvas 开发的炫酷坦克大战网页游戏。

## 技术栈

- **Vue 3.5** - 最新版本的 Vue.js 框架
- **TypeScript** - 类型安全的 JavaScript
- **Vite** - 快速的前端构建工具
- **Canvas API** - 高性能 2D 渲染
- **OxLint** - 快速的代码检查工具

## 功能特性

### 游戏核心
- 🎮 玩家坦克控制（WASD/方向键移动，空格键射击）
- 👾 多种敌人类型（普通、快速、重型）
- 🧱 多种地形障碍（砖墙、钢墙、水域）
- 💥 粒子爆炸效果
- ✨ 炫酷的视觉特效

### 关卡系统
- 📊 6 个精心设计的关卡
- 📈 难度递增的敌人数量和速度
- 🏆 关卡完成奖励

### 记分系统
- 🎯 击杀敌人获得分数
  - 普通敌人：100 分
  - 快速敌人：200 分
  - 重型敌人：300 分
- 💎 道具收集加分

### 道具系统
- ⚡ 速度提升 - 增加移动速度
- ★ 火力增强 - 减少射击冷却
- ♥ 生命恢复 - 增加一条生命
- ◆ 护盾加分 - 直接获得 50 分

### 视觉效果
- 🌈 渐变色 UI 设计
- 💫 发光特效和阴影
- 🎆 爆炸粒子系统
- 🌊 动态水波效果
- 🎨 炫酷的动画过渡

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

游戏将在 http://localhost:3000 启动

### 构建生产版本

```bash
npm run build
```

### 代码检查

```bash
npm run format
```

## 游戏操作

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| A / ← | 向左移动 |
| S / ↓ | 向下移动 |
| D / → | 向右移动 |
| 空格 | 射击 |
| P | 暂停/继续 |

## 游戏规则

1. 消灭所有敌人即可进入下一关
2. 被敌人子弹或敌人本体击中会损失一条生命
3. 生命值归零时游戏结束
4. 收集道具可以获得各种增益效果
5. 钢墙无法被破坏，水域可以通行但子弹会穿过

## 开发说明

### 游戏核心类

- **Player** - 玩家坦克
- **Enemy** - 敌人坦克
- **Bullet** - 子弹
- **Wall** - 墙壁障碍物
- **Particle** - 粒子效果
- **PowerUp** - 道具

### 游戏循环

游戏使用 `requestAnimationFrame` 实现流畅的 60fps 渲染循环：
1. `update()` - 更新游戏状态
2. `draw()` - 绘制游戏画面
